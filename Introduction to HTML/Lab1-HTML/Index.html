    Introduction

    A web page being render in the browser of many thing - logo, informative text,picture,hyperlinks,navigational structure and more. HTML5 offers a set of markup elements that allow you to create a structured layout for web pages. These elements are often termed as Semantic Markup because they convey their meaning and purpose clearly to the developer and the browser. This article discusses some of the important HTML5 elements that can contribute to the Layout of web page.

    Semantic Markup

    HTML5 includes a set of mark up elements that overcome this difficulty. These new elements have meaningful names so that just by look at these elements you got a clear idea about their content.These semantic elements of HTML5 are listed below(this in no an exhaustive list):

    1. Header
    2. footer
    3. section
    4. aside
    5. nav
